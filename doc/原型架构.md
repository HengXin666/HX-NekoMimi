# HX-NekoMimi
## 原型架构
### 1. 项目功能
核心目的: 听书

输入:
    - mp3 书籍音频
    - mp3 书籍字幕 (SRT / ASS)

需要功能:
    - 书籍管理 (书籍列表、书籍详情)
    - 书籍播放 (播放、暂停、快进、快退)
    - 自动记忆播放进度 (上次播放的书籍、章节、进度)

使用方法:

[打开] -> 主界面: 显示封面 or 书籍名称卡片
            -> 打开书详情页 (如下):
                |    书籍名称     |
                | > 上次播放位置 < |
                | -------------  |
                | List Item 1    |
                | List Item 2    |
                | List Item 3    |
                |                |
                -> 1. 可看到上次记忆的播放文件和位置 (如上显示; 如果没有就不显示)
                -> 2. 显示书籍所有 章节列表 (根据 mp3 文件名生成 (如果有文件夹则保留对应树结构))
                    -> 点击书籍章节, 播放对应章节
                      -> 播放页面
                         -> 显示章节名称 (mp3 文件名)
                         -> 显示字幕 (SRT / ASS)
                         -> 显示 上次播放位置 (如上显示; 如果没有就不显示) [可以跳转]
                         -> 显示进度条
                         -> 显示播放按钮 (播放、暂停、快进、快退)
                         -> [后台功能: 自动记忆播放进度]
                -> 3. 可以刷新章节列表 (重新扫描 mp3 文件 (从系统目录))
            -> 删除书籍
            -> 添加书籍
                -> 1. 选择书籍文件(夹)
                -> 2. 递归扫描 mp3 文件 (从系统目录) 进行添加
            -> 刷新书籍的章节列表
